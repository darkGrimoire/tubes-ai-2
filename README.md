# tubes-ai-2
Tubes AI 2: CLIPS Exploration



-------
## Requirements

- Python 3.x
- pyqt5 (for GUI version)
- clipspy

-------
## Usage

main.py > Minesweeper AI console version  
app.py > Minesweeper AI GUI version (unstable)

-------
### Run console version
Run in terminal:
```bash
python main.py
```
### Run GUI version
Run in terminal:
```bash
python app.py
```
